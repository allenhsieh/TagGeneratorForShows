# Tags for social media and Archive.org new item link generator
I was sick of manually typing tags for [the videos](https://archive.org/details/@djpandaexpress) I shoot.

This app will now also generate a [link with preset metadata](https://blog.archive.org/2013/02/08/presetting-metadata-with-the-new-beta-uploader/) based on selected tags for new Archive.org items
